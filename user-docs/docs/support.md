# Support

Get help with Communities.

## Community Support

Your first stop for help should be your community:

### Community Forum

Most communities have forums where you can:
- Ask questions about using Communities
- Get help from experienced members
- Learn community-specific norms and practices
- Report issues you're experiencing

**How to access**: Go to your community → **Forum** → Post your question

### Community Admins

Your community administrators can help with:
- Account and permission issues
- Community-specific settings
- Conflicts or disputes
- Membership questions

**How to find admins**: Go to **Community** → **Members** → Filter by "Admin" role

## Technical Support

### Reporting Bugs

If you've found a technical problem:

1. **Check if it's known**: Search existing bug reports
2. **Gather information**:
   - What were you trying to do?
   - What happened instead?
   - What browser/device are you using?
   - Screenshots if possible
3. **Report it**:
   - To your community admin (they can escalate)
   - To the platform maintainers (if you have access)

### Feature Requests

Have an idea for improving Communities?

1. **Check if it exists**: Search documentation and forums
2. **Discuss with your community**: Get feedback from members
3. **Propose to admins**: Community admins can request features
4. **Submit formally**: If there's a feature request process

## Self-Help Resources

### Common Issues and Solutions

#### Can't Log In

**Problem**: Login fails or shows error

**Solutions**:
1. Verify email and password are correct
2. Check if your email is verified (check inbox/spam)
3. Try password reset (click "Forgot Password")
4. Clear browser cache and cookies
5. Try a different browser
6. Contact your community admin

#### Missing Permissions

**Problem**: Can't access a feature you think you should have

**Solutions**:
1. Check your current trust score (Profile → Trust)
2. Check required threshold (Community Settings → Trust Thresholds)
3. Verify the feature is enabled (ask admin)
4. Check if you need a specific role (not just trust)
5. Wait for the page to refresh (permissions update in real-time)

#### Not Receiving Notifications

**Problem**: Expected notifications aren't appearing

**Solutions**:
1. Check notification settings (Profile → Settings → Notifications)
2. Refresh the page (notifications load periodically)
3. Check browser notification permissions (for web notifications)
4. Verify the feature generates notifications (some don't)
5. Check your activity feed (Dashboard → Activity)

#### Can't Share Resources

**Problem**: Unable to create wealth to share

**Solutions**:
1. Check trust threshold for sharing (usually 10)
2. Verify community allows wealth sharing (feature flag)
3. Ensure item type exists or you can create custom items
4. Try refreshing the page
5. Check for browser console errors

#### Pool/Council Not Working

**Problem**: Pool distribution or council features don't work

**Solutions**:
1. Verify you're a manager of that pool/council
2. Check community trust thresholds for pool/council features
3. Ensure you're looking at the right community
4. Try refreshing the page
5. Contact community admin

- **Chrome/Chromium**: Version 100+

- **Safari**: Version 15+

Older browsers may have issues. Update your browser if experiencing problems.

- Android Firefox (latest)

Native apps may be developed in the future.

## Privacy and Security

### Reporting Security Issues

If you've found a security vulnerability:

**DO NOT** post it publicly.

Instead:
1. Contact your instance administrator privately
2. Provide details about the vulnerability
3. Allow time for a fix before disclosing

### Privacy Concerns

If you have privacy concerns:
1. Review your [privacy settings](getting-started/creating-account.md#configure-privacy-settings)
2. Contact your community admin about community privacy settings
3. Review the platform privacy policy (if available)

### Data Requests

To request your data or deletion:
1. Go to **Profile Settings** → **Privacy**
2. Request data export (if available)
3. Request account deletion (if desired)
4. Contact admin if you need help

## Community Health

### Reporting Harmful Behavior

If you experience or witness harmful behavior:

1. **Immediate safety**: Remove yourself from the situation
2. **Documentation**: Save screenshots/evidence if safe to do so
3. **Report**:
   - Flag forum content (if appropriate)
   - Contact community admins
   - Create a dispute (for wealth-related issues)
   - Seek external support if needed
4. **Support**: Reach out to trusted community members

### Conflict Resolution

For interpersonal conflicts:
1. Try [direct communication](features/disputes.md#before-creating-a-dispute) first
2. Seek informal mediation from trusted members
3. Use the [dispute system](features/disputes.md) for formal mediation
4. Contact admins for serious issues

### Mental Health Resources

Community participation can be emotionally complex. If you're struggling:
- Take breaks when needed
- Set boundaries on your participation
- Seek support from friends, family, or professionals
- Remember: leaving or taking space is okay
